#  Nyawitngoding - UI/UX Concept
<img width="150" height="90" alt="logo-sawitngoding" src="https://github.com/user-attachments/assets/15d3fcdf-0271-4046-84cb-8a5a706b9f36" />

**Nyawitngoding** adalah konsep aplikasi edukasi teknologi yang dirancang untuk membantu pengguna menentukan jalur belajar (*learning path*) di dunia IT dengan antarmuka yang bersih dan interaktif.

## 🚀 Alur Aplikasi (Flow)
1. **Splash Screens**: Perkenalan visual aplikasi (4 tahap).
2. **Authentication**: 
   - Sign In & Sign Up.
   - Forgot Password & OTP Verification.
3. **Main Dashboard**: Beranda utama pengguna.
4. **Learning Path**: Alur pembelajaran atau modul interaktif.
5. **Contact Me**: Fitur dukungan atau komunikasi.

### Perangkat Target
- **Device Frame**: iPhone 16 & 17 Pro Max
- **Tema**: Light Mode dengan aksen hijau alami.
